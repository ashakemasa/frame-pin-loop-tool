# Frame Pin Loop Tool

動画からピン指定したフレームを抽出し、番号順にループ確認する制作ツールです。

## GitHub Pages

このフォルダの中身をリポジトリ直下に置き、GitHub Pages の Source を `Deploy from a branch`、Branch を `main`、Folder を `/root` に設定してください。

公開後は `https://<user>.github.io/<repo>/` で開けます。

## iPad

1. iPad の Safari で公開URLを開く
2. 共有ボタンを押す
3. `ホーム画面に追加` を選ぶ

動画ファイルは端末内で処理され、サーバーへアップロードされません。
